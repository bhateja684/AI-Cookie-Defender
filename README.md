# AI Cookie Defender

> Intelligent Browser Extension for AI-Assisted Detection of Malicious and Tracking Cookies

---

## Overview

AI Cookie Defender is a Manifest V3 browser extension that continuously analyzes HTTP cookies generated by visited websites and identifies potentially malicious, invasive or tracking-oriented cookies using a hybrid AI-assisted detection pipeline.

Unlike traditional cookie viewers that only enumerate stored cookies, AI Cookie Defender evaluates each cookie using structural, behavioral and security-related features before assigning a threat score.

All analysis is performed locally within the browser, ensuring that browsing data never leaves the user's device.

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/bhateja684/AI-Cookie-Defender.git
```
or download the repository as a ZIP file and extract it.

### 2. Open Chrome Extensions

Navigate to:

```text
chrome://extensions/
```

### 3. Enable Developer Mode

Turn on **Developer Mode** using the toggle in the top-right corner.

### 4. Load the Extension

Click **Load unpacked** and select the project folder containing `manifest.json`.

### 5. Start Using AI Cookie Defender

Pin the extension from the Extensions menu and click the cookie icon to analyze cookies on the active website.


## Detection Strategy

The detection engine evaluates multiple characteristics of every cookie, including:

* Domain reputation
* Tracking signatures
* Cookie entropy
* Base64-encoded payloads
* Cookie lifetime analysis
* Third-party scope
* Secure and HttpOnly attributes
* Behavioral heuristics
* AI confidence scoring
